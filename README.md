# Pokémon Project: Pi

**Pokémon Project: Pi** is a ROM hack of **Pokémon FireRed (USA) v1.0** *(1636 – Fire Red Squirrels)*.

At its core, Project: Pi is a complete rebalance hack built around an entirely new **175-Pokémon regional Pokédex**, with every Pokémon obtainable in a single playthrough. Alongside the new Pokédex, the game features numerous quality-of-life improvements, graphical enhancements, and modern gameplay mechanics.

One of the project's primary goals is to integrate newer Pokémon into Kanto as naturally as possible, creating an experience that feels faithful to the original **Pokémon FireRed** while offering a fresh adventure.

The game is complete through the **Elite Four** and includes an additional **post-game event**.

**Project: Pi is not intended to be a difficulty hack**. Instead, it focuses on providing a balanced and consistently fair experience through carefully designed encounters, improved progression, and badge-based level caps.
